# WeatherInsight

## Описание
WeatherInsight — это интерактивное веб-приложение для анализа погодных данных.  
Проект включает:
- Автоматический пайплайн загрузки данных (Apache Airflow)
- Генерацию тестовых погодных данных для 20 городов
- Интерактивный дашборд на Streamlit с графиками и фильтрами

## Функциональность
- 📊 **20 городов** (Москва, Лондон, Токио, Нью-Йорк, Париж, Пекин, Берлин, Стамбул, Мадрид, Рим, Каир, Дубай, Сидней, Лос-Анджелес, Сингапур, Сеул, Бангкок, Рио-де-Жанейро, Кейптаун, Торонто)
- 📅 **Фильтр по датам** (выбор диапазона)
- 🌡️ **Графики:**
  - Динамика температуры (линейный)
  - Осадки (столбчатая диаграмма)
  - Boxplot температуры по городам
  - Гистограмма температуры
  - Сравнение средней температуры по городам
  - Скользящее среднее (7 дней)
  - Корреляционная матрица
  - Индекс комфорта (факультативно)
- 📋 **Таблица данных** с сортировкой
- ⏰ **Автоматическое обновление** данных каждый час (Airflow)

## Установка и запуск

### 1. Клонируйте репозиторий
```bash
git clone https://github.com/ваш_ник/weather_insight.git
cd weather_insight
2. Запустите Airflow (опционально, для автоматической загрузки)
bash
docker-compose up -d
Airflow будет доступен по адресу: http://localhost:8080 (логин: admin, пароль: admin)

3. Сгенерируйте тестовые данные (если не используете Airflow)
bash
python scripts/fetch_and_store.py
4. Запустите Streamlit-приложение
bash
streamlit run app.py
Зависимости
bash
pip install streamlit pandas plotly polars sqlite3
Структура проекта
text
